# Jailbreak Attack Evaluation

This repository contains the code, experimental results, and demonstration videos for conducting jailbreak attack evaluations on four large language models discussed in the paper.

## Overview

The project provides a tutorial pipeline for running jailbreak attack tests and includes the corresponding experimental outputs. Demonstration videos are also provided to show the jailbreak attack process on each evaluated model.

## Running the Attack Test

To execute the jailbreak attack testing pipeline, run:
